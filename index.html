Aprendiendo HTML5

Esta página está dedicada a explicar qué es HTML5, su estructura, sus etiquetas y su importancia en el desarrollo web. El contenido será completado por el alumno.

Comenzar

¿Qué es HTML5?

HTML5 es la versión más reciente del lenguaje de marcado HTML, utilizado para crear páginas web. Permite estructurar el contenido de forma clara y moderna, además de incluir nuevas funciones como video, audio y gráficos sin necesidad de programas externos. Es fundamental en el desarrollo web actual porque facilita la creación de sitios más rápidos, accesibles y compatibles con diferentes dispositivos.

Estructura básica de HTML5

<!DOCTYPE html> <html> <head> <title>Mi página</title> </head> <body> </body> </html>

La estructura básica de HTML5 está formada por varias etiquetas principales. El DOCTYPE indica que se trata de un documento HTML5. La etiqueta html contiene todo el contenido de la página. Dentro de ella, head incluye información como el título y metadatos, mientras que body contiene todo lo visible para el usuario. Esta estructura permite organizar correctamente una página web.

Etiquetas principales

h1 - h6

Las etiquetas h1 a h6 se utilizan para definir títulos y subtítulos. h1 es el título más importante y h6 el menos importante. Ayudan a organizar el contenido y mejorar la lectura.

p

La etiqueta p se utiliza para escribir párrafos de texto. Es una de las más usadas en HTML porque permite mostrar información de manera clara y ordenada.

img

La etiqueta img permite insertar imágenes en una página web. Utiliza atributos como src para la ruta de la imagen y alt para describirla.

a

La etiqueta a se utiliza para crear enlaces que permiten navegar entre páginas o sitios web. Se utiliza el atributo href para indicar la dirección del enlace.

button

La etiqueta button crea botones interactivos que pueden ejecutar acciones, como enviar formularios o activar funciones con JavaScript.

ul / ol / li

Las etiquetas ul y ol permiten crear listas. ul es para listas sin orden y ol para listas ordenadas. li se utiliza para cada elemento de la lista.

HTML5 semántico

header

La etiqueta header representa el encabezado de la página o de una sección. Generalmente contiene el título, logo o menú de navegación.

nav

La etiqueta nav se utiliza para definir un menú de navegación con enlaces hacia otras partes del sitio.

main

La etiqueta main contiene el contenido principal del documento. Solo debe haber una por página.

section

La etiqueta section agrupa contenido relacionado dentro de una página, como diferentes temas o apartados.

article

La etiqueta article representa contenido independiente, como una noticia, publicación o artículo.

footer

La etiqueta footer representa el pie de página, donde se coloca información como autor, derechos o datos de contacto.

Formularios en HTML5

Los formularios en HTML5 permiten recopilar información del usuario, como nombres, correos o comentarios. Se utilizan etiquetas como form para crear el formulario, label para los textos descriptivos, input para campos de entrada, textarea para textos largos y button para enviar la información.

Multimedia en HTML5

HTML5 permite integrar contenido multimedia como imágenes, audio y video directamente en la página web. La etiqueta img muestra imágenes, audio permite reproducir sonido y video permite mostrar videos. Además, el atributo alt es importante para describir imágenes y mejorar la accesibilidad.

Imágenes

La etiqueta img se utiliza para insertar imágenes en una página web. Funciona con el atributo src para indicar la ruta de la imagen y alt para describirla. Es importante optimizar las imágenes para que carguen rápido.

Audio

La etiqueta audio permite reproducir sonidos directamente en la página web. Puede incluir controles como reproducir, pausar y ajustar volumen. Es compatible con formatos como MP3 y WAV.

Video

La etiqueta video permite mostrar videos sin necesidad de plugins externos. Incluye controles para reproducir, pausar y avanzar el contenido. Es compatible con formatos como MP4.

Accesibilidad

La accesibilidad en HTML5 es importante para que todas las personas puedan usar la web. El atributo alt en las imágenes describe su contenido, lo que ayuda a personas con discapacidad visual y mejora el posicionamiento en buscadores.

Conclusión

HTML5 es una herramienta fundamental para el desarrollo web moderno, ya que permite crear sitios más organizados, accesibles e interactivos. Gracias a sus nuevas etiquetas y funciones, facilita el trabajo de los desarrolladores y mejora la experiencia de los usuarios. Aprender HTML5 es esencial para cualquier persona que desee crear páginas web.
